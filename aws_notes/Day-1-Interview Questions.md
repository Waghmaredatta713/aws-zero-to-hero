# Q1: What is cloud computing?
Cloud computing is the on-demand delivery of computing resources over the internet with pay-as-you-go pricing.

# Q2: What are the advantages of cloud?
Cost-effective, scalable, high availability, disaster recovery, global access.

# Q3: What is a region?
A region is a geographical area where AWS has multiple data centers.

# Q4: What is an Availability Zone?
An AZ is a physically separate data center within a region.

# Q5: What is high availability?
High availability ensures your application is always accessible even if failures occur.

# Q6: What is scalability?
Scalability is the ability to increase or decrease resources based on demand.

# Q7: What is elasticity?
Elasticity is automatic scaling up and down of resources.

# Q8: What is fault tolerance?
Fault tolerance means system continues to operate even when components fail.

# Q9: What is disaster recovery?
Disaster recovery is the process of restoring systems after a failure.

# Q10: Why is AWS popular?
Because it is scalable, secure, global, and pay-as-you-go.
